# abi-convert
```
Usage: abi-manager [options] [command]

CLI to manage ABI files

Options:
  -V, --version                            output the version number
  -v, --verbose                            show verbose logging (default: false)
  -h, --help                               display help for command

Commands:
  convert [options] <abiPath>              convert an ABI file
  batch-convert [options] <directoryPath>  convert a directory of ABI files
  help [command]                           display help for command

```
